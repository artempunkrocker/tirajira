# TiraJira (тиражира) - Автоматизация создания задач в Jira

[![Версия PyPI](https://badge.fury.io/py/tirajira.svg)](https://badge.fury.io/py/tirajira)
[![Лицензия: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

**Устали от ручного создания десятков и сотен задач в Jira?** TiraJira - ваше решение!

TiraJira - это мощная утилита командной строки для автоматизации массового создания задач в Jira. Просто подготовьте задачи в файле (JSON, YAML, CSV, Excel или XML), и TiraJira сделает все остальное - создаст все ваши задачи за считанные минуты вместо часов ручной работы.

## Ключевые особенности

- 🚀 **Массовое создание задач** - создавайте десятки и сотни задач за одну операцию
- 📁 **Поддержка разных форматов** - используйте привычные форматы JSON, YAML, CSV, Excel или XML
- 🔗 **Автоматическая привязка к эпикам** - связывайте задачи с эпиками без дополнительных действий
- 🔗 **Создание связей между задачами** - автоматически создавайте связи между новыми и существующими задачами
- ⚡ **Пакетная обработка** - оптимизированная отправка задач для максимальной производительности
- 🛠️ **Полная поддержка кастомных полей** - используйте все поля вашего Jira, включая пользовательские
- 📊 **Подробное логирование** - отслеживайте процесс создания каждой задачи в реальном времени
- 🛑 **Режим остановки при ошибках** - прекращение обработки при возникновении ошибок (--stop-on-error)
- 📋 **Генерация отчетов** - получайте машиночитаемые отчеты в форматах JSON, CSV, Excel, YAML, XML
- 🔁 **Возобновление выполнения** - продолжайте работу с места остановки при возникновении ошибок
- 🔒 **Защита от directory traversal атак** - предотвращение доступа к файлам вне рабочей директории

## Быстрый старт

1. Установите TiraJira:
   ```bash
   pip3 install tirajira
   ```

2. Подготовьте файл с задачами в формате JSON:
   ```json
   [
     {
       "project": {"key": "PROJ"},
       "summary": "Первая задача",
       "description": "Описание первой задачи",
       "issuetype": {"name": "Task"}
     }
   ]
   ```

3. Создайте задачи в Jira:
   ```bash
   tirajira import tasks.json
   ```

Готово! Ваши задачи созданы в Jira.

## Системные требования

- Python 3.10 или выше
- Доступ к Jira Server
- API-токен для доступа к Jira

## Установка

Для обычных пользователей (рекомендуется):
```bash
pip3 install tirajira
```

Для разработчиков:
```bash
git clone https://github.com/ваш-логин/tirajira.git
cd tirajira
poetry install
```

[Подробнее об установке](installation.md)

## Настройка подключения к Jira

TiraJira поддерживает два режима аутентификации:
- **Jira Cloud** - Basic Auth (email + API token)
- **Jira Server/Data Center** - Personal Access Token

[Подробнее о настройке подключения](configuration.md)

## Подготовка файла с задачами

Поддерживаются форматы: JSON, YAML, CSV, Excel, XML

Минимальный пример (JSON):
```json
[
  {
    "project": {"key": "PROJ"},
    "summary": "Моя первая задача",
    "issuetype": {"name": "Task"}
  }
]
```

[Подробнее о форматах файлов](file_formats.md)

## Команды

- `import` - Создание задач из файла
- `resume` - Возобновление выполнения из отчета
- `extract-failed` - Извлечение неудачных задач из отчета
