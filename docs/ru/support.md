## Получение помощи

### Проблемы с установкой

Если у вас возникли проблемы с установкой TiraJira:

1. Убедитесь, что у вас установлен Python 3.10 или выше:
   ```bash
   python3 --version
   ```

2. Проверьте, установлен ли Poetry:
   ```bash
   poetry --version
   ```

3. Если возникают ошибки при установке зависимостей, попробуйте:
   ```bash
   pip3 install --upgrade pip
   pip3 install tirajira
   ```

### Проблемы с подключением к Jira

Если TiraJira не может подключиться к Jira:

1. Проверьте правильность URL Jira в конфигурации
2. Убедитесь, что учетные данные верны
3. Проверьте, есть ли у пользователя права на создание задач
4. Убедитесь, что API доступен и не заблокирован брандмауэром

### Проблемы с форматами файлов

Если задачи не создаются из-за проблем с форматом файла:

1. Проверьте формат файла на соответствие примерам в документации
2. Убедитесь, что файл не содержит синтаксических ошибок
3. Попробуйте начать с минимального примера и постепенно добавлять сложность
4. Используйте флаг `--verbose` для получения подробной информации об ошибках

## Устранение неполадок

### Проверка версии

Проверьте установленную версию TiraJira:
```bash
tirajira --version
```

### Подробное логирование

Используйте флаг `--verbose` для получения подробной информации:
```bash
tirajira import tasks.json --verbose
```

### Проверка конфигурации

Убедитесь, что переменные окружения установлены правильно:
```bash
# Для Jira Cloud
echo $JIRA_SERVER
echo $JIRA_USER
echo $JIRA_API_TOKEN

# Для Jira Server
echo $JIRA_SERVER
echo $JIRA_TOKEN
```

## Сообщение об ошибках

При сообщении об ошибках обязательно предоставьте:

1. Версию TiraJira (`tirajira --version`)
2. Версию Python (`python3 --version`)
3. Операционную систему
4. Полные шаги для воспроизведения ошибки
5. Файл с задачами (без конфиденциальной информации)
6. Полный текст ошибки
7. Логи выполнения (с флагом `--verbose` при возможности)
